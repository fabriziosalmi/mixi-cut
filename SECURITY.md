# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | ✅        |
| < 0.2   | ❌        |

## Reporting a Vulnerability

MIXI-CUT is an audio signal generator — it does not handle network traffic, authentication, or sensitive data. However, if you discover a security issue (e.g., in the WAV parser, file handling, or dependency chain), please report it responsibly:

1. **Email**: fabrizio.salmi@gmail.com
2. **Subject**: `[SECURITY] MIXI-CUT: <brief description>`
3. **Timeline**: We aim to respond within 48 hours

Please **do not** open a public GitHub issue for security vulnerabilities.
